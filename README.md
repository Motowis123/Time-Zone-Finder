# Time-Zone-Finder
Finds your timezone
Install pip if you havent by typing this in your cmd:
python get-pip.py
And intall tzlocal:
pip install tzlocal
Requires Python, install python here: https://www.python.org/ftp/python/3.12.4/python-3.12.4-amd64.exe
PASTE LINK IN BROWSER TO GET RIGHT VERSION
